Achte auf das Passiv

In den Dialogen: „Termine werden eingehalten", „Fremde werden gesiezt", „„Guten Appetit" wird gesagt". Das Passiv = werden + Partizip II. Wer es tut, ist oft unwichtig — genau das üben wir gleich.

⚙️ Grammatik: Passiv (Präsens)

Wenn die Handlung wichtiger ist als die Person, die sie tut.

Die Regel: Passiv Präsens = werden (konjugiert) + Partizip II (am Satzende).
Aktiv: „Man schmückt den Baum." → Passiv: „Der Baum wird geschmückt."
Aktiv: „Man sagt „Guten Appetit"." → Passiv: „„Guten Appetit" wird gesagt."
Plural: „Termine werden eingehalten." (werden + Partizip II)
AktivPassiv (Präsens)
Man schmückt den Baum.Der Baum wird geschmückt.
Man feiert das Fest.Das Fest wird gefeiert.
Man siezt Fremde.Fremde werden gesiezt.
Man hält Termine ein.Termine werden eingehalten.
