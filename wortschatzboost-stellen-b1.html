Der Trick mit den Vorsilben

Ein Verb, viele Bedeutungen — nur durch die Vorsilbe!

📥 So funktioniert's: Der Stamm stellen heißt to put / place. Setzt du eine Vorsilbe davor, ändert sich die Bedeutung komplett.

Quiz: Welches Wort?

Tipp auf die richtige Antwort.

🛒 „Im Restaurant ___ ich mir meistens eine Pizza." (ordern)
bestellestelle … vorstelle … her
✓ bestellen = in Auftrag geben (untrennbar).
🎤 „Darf ich dir meinen Kollegen ___?" (bekannt machen)
bestellenvorstellenfeststellen
✓ jemanden vorstellen = bekannt machen (trennbar).
🏭 „Diese Fabrik ___ moderne Möbel ___." (produzieren)
stellt … herstellt … vorbestellt
✓ herstellen = produzieren (trennbar).
🔍 „Der Arzt ___ eine leichte Erkältung ___." (diagnostizieren)
stellt … vorstellt … festbestellt
✓ feststellen = bemerken / diagnostizieren (trennbar).

eingestelltbestelltvorgestellt
✓ einstellen = anstellen / einstellen (trennbar).
