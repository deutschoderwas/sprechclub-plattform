Das brauchst du: die wichtigsten Begriffe

Lies das VOR der Stunde. Dann verstehst du im Unterricht sofort, worum es geht.

🙋
das Aktiv
Wer macht etwas? → „Der Mechaniker repariert das Auto."
🔧
das Passiv
Was passiert mit etwas? Die Handlung zählt. → „Das Auto wird repariert."
⚙️
das Vorgangspassiv
werden + Partizip II — etwas passiert gerade. „Die Tür wird geöffnet."
📦
das Zustandspassiv
sein + Partizip II — das Ergebnis. „Die Tür ist geöffnet."
👤
das Agens
Wer/was es tut: von + Dativ (Person), durch + Akkusativ (Mittel/Ursache).
🧩
das Partizip II
gemacht, repariert, geschrieben, gebracht … — die Form für Perfekt und Passiv.
Kernidee: Im Passiv ist die Handlung wichtig, nicht die Person. Deshalb sagt man oft gar nicht, wer es tut: „Hier wird gebaut.", „Das Formular wird geprüft."

Die Bildung – ganz einfach

Passiv = eine Form von werden + Partizip II (am Satzende).

1. Vom Aktiv zum Passiv

Das Objekt (Akkusativ) im Aktiv wird zum Subjekt im Passiv.

AktivPassiv
Der Arzt untersucht den Patienten.Der Patient wird untersucht.
Man baut die Brücke.Die Brücke wird gebaut.

✔ Das Auto wird repariert.   Das Auto wird reparieren.

2. Agens: von oder durch?

von + Dativdurch + Akkusativ
Person / Verursacher
„Der Brief wird von der Chefin geschrieben."		Mittel / Ursache
„Die Stadt wurde durch das Hochwasser zerstört."

3. Passiv mit Modalverb

Modalverb (Position 2) + Partizip II + werden (Infinitiv) am Ende.

✔ Das Formular muss heute abgegeben werden.

✔ Hier darf nicht geraucht werden.

Passiv in allen Zeiten

Immer Partizip II — nur die Form von werden ändert sich. Beispielverb: reparieren → repariert.

ZeitForm von „werden"Beispiel
PräsenswirdDas Auto wird repariert.
PräteritumwurdeDas Auto wurde repariert.
Perfektist … wordenDas Auto ist repariert worden.
Plusquamperfektwar … wordenDas Auto war repariert worden.
Futur Iwird … werdenDas Auto wird repariert werden.
mit Modalverbmuss … werdenDas Auto muss repariert werden.
Merke: Im Perfekt heißt es worden (nicht „geworden"): „Es ist gemacht worden."

Quiz: Was ist richtig?

Tippe die richtige Antwort an.

Aktiv: „Man liest das Buch." → Passiv (Präsens):
Vorgangspassiv = werden + Partizip II (gelesen).
Perfekt Passiv: „Das Haus ___ letztes Jahr gebaut ___."
Perfekt Passiv: sein + Partizip II + worden.
Welche Präposition? „Der Roman wurde ___ einer berühmten Autorin geschrieben."
Person = von + Dativ.
Passiv mit Modalverb: „Die Rechnung ___ bis Freitag bezahlt werden."
Modalverb + Partizip II + werden: „muss bezahlt werden".

Lückentext: werden in der richtigen Form

Wähle aus jeder Liste die passende Form.

1. Die Pizza in 20 Minuten geliefert. (Präsens)
2. Das Konzert wegen des Regens abgesagt. (Präteritum)
3. Der Vertrag ist gestern unterschrieben . (Perfekt)
4. Das Paket muss heute noch abgeschickt . (Modalverb)
5. Die Ergebnisse dem Lehrer korrigiert. (Agens: Person)

Aktiv ↔ Passiv zuordnen

Tippe links einen Aktivsatz an, dann rechts das passende Passiv.

Aktiv

Die Firma stellt neue Mitarbeiter ein.
Ein Sturm zerstörte das Dach.
Man muss das Formular ausfüllen.
Der Koch hat die Suppe gekocht.

Passiv

Das Formular muss ausgefüllt werden.
Neue Mitarbeiter werden eingestellt.
Die Suppe ist gekocht worden.
Das Dach wurde durch einen Sturm zerstört.
