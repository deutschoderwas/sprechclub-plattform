Der Trick mit den Vorsilben

Ein Verb, viele Bedeutungen — nur durch die Vorsilbe!

🧭 So funktioniert's: Der Stamm führen heißt to lead / guide. Setzt du eine Vorsilbe davor, ändert sich die Bedeutung komplett — und du musst wissen, ob sie trennbar oder untrennbar ist.
Trennbar (Vorsilbe ans Ende)Untrennbar (bleibt zusammen)
an-, aus-, durch-, ein-, auf-, vor-, ab-, zurück-, weiter-, mit-

Quiz: Welches Wort?

Tipp auf die richtige Antwort.

📊 „Nächste Woche ___ wir eine große Umfrage ___."
führen … durchführen … ausführen … an
✓ durchführen = veranstalten, conduct (trennbar).
🆕 „Das Unternehmen will ein neues Produkt ___." (auf den Markt bringen)
einführenabführenverführen
✓ einführen = neu auf den Markt bringen.
😈 „Der Rabatt hat mich zum Kaufen ___."
vorgeführtverführtangeführt
✓ verführen = in Versuchung bringen (untrennbar).
⚙️ „Der Computer ___ das Programm sofort ___."
führt … ausführt … aufführt … an
✓ ausführen = einen Befehl erledigen (trennbar).
↩️ „Der Unfall lässt sich auf Glatteis ___."
zurückführenweiterführenhinführen
✓ zurückführen auf = als Ursache erklären.
🚔 „Die Beweise ___ den Täter."
überführenvorführenabführen
✓ überführen = jemandem die Tat beweisen (untrennbar).
